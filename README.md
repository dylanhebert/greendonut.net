# greendonut.net

Personal portfolio site for [Green Donut](https://greendonut.net). Showcases software projects, video game dev, and other creations.

Built with Flask, Tailwind CSS, and vanilla JavaScript.

## Features

- Multi-theme system (Light, Dark, Retro, etc.) with system preference detection
- Music player with waveform visualization, frequency bars, and oscilloscope modes
- Animated background blobs and scroll-reveal animations
- Nav scroll animation with centered links and sliding brand
- Dead High rotating screenshot slideshow with logo hover effect
- Responsive design
