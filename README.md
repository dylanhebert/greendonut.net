# greendonut.net

Personal portfolio site for [Green Donut](https://greendonut.net). Showcases software projects, video game mods, and other creations.

Built with Flask, Tailwind CSS, and vanilla JavaScript.

## Features

- Dark/light mode with system preference detection
- Animated background blobs and scroll-reveal animations
- Nav scroll animation with centered links and sliding brand
- Dead High rotating screenshot slideshow with logo hover effect
- Responsive design
